# 运行环境
- Windows 11（amd）；其他平台应该问题也不大
- python-3.13.3 ；python3 应该比较新的版本都可以
- 第三方库依赖：需要一句【pip install flask flask-cors numpy scikit-learn sentence-transformers GitPython】，预留3-4GB或以上的磁盘空间较好

# 本地运行步骤
1. 执行python app.py，等待模型初始化完成
2. 用浏览器打开index.html

# 预估成本
2.5 $（假设一次生成就满足需求的话）
